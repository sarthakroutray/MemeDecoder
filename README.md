# Meme Decoder

A retro terminal-themed meme identification game built with React, TypeScript, and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your meme images to the `public/memes/` directory (named 2.jpg, 3.jpg, etc.)

3. Run the development server:
```bash
npm run dev
```

4. Open your browser to the local URL shown in the terminal

## How to Play

- Identify blurred memes within 60 seconds
- Type to fill in the answer slots
- Every 30 seconds, a hint letter is revealed
- Wrong answers reduce blur (making it easier)
- Complete all 4 memes before time runs out!

## Build for Production

```bash
npm run build
```
